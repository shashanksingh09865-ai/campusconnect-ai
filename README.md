# CampusConnect AI

CampusConnect AI is an AI-powered college management system built using FastAPI, SQLite, SQLAlchemy, JWT Authentication, and Google Gemini AI.

## Features

* User Registration & Login
* JWT Authentication
* Role-Based Access Control
* Notice Management
* Notes Management
* PDF Upload System
* AI PDF Summarization
* AI Chat Assistant
* Dashboard API

## Technologies Used

* Python
* FastAPI
* SQLite
* SQLAlchemy
* JWT
* Google Gemini AI
* Pydantic

## Run Project

```bash
python -m uvicorn main:app --reload
```
